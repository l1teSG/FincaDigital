# 🐄 FincaDigital

> **Ecosistema integral para el control de producción, gastos y salud bovina.**
> Una solución tri-plataforma que conecta el campo con la oficina mediante Web, Mobile y Excel.

---

## 📌 Visión del Proyecto
Llevar un control preciso en la ganadería suele ser difícil por la falta de conectividad en el campo. **Bovino360** soluciona esto permitiendo la captura de datos rápida en dispositivos móviles, la visualización estratégica en la web y el análisis profundo en hojas de cálculo.

## 🛠️ Stack Tecnológico

Este proyecto utiliza un enfoque moderno y escalable:

* **Mobile App:** [React Native](https://reactnative.dev/) + [Expo](https://expo.dev/) (Ideal para captura rápida y soporte offline).
* **Web Dashboard:** [Astro](https://astro.build/) o [React](https://reactjs.org/) + [Tailwind CSS](https://tailwindcss.com/) (Para reportes rápidos y fluidos).
* **Backend & DB:** [Supabase](https://supabase.com/) (PostgreSQL + Auth + Realtime).
* **Data Analysis:** Conector para **Microsoft Excel** vía Power Query / CSV para auditorías contables.

---

## 🚀 Características Principales

### 💰 Control Financiero (Gastos vs Ingresos)
* **Gastos de Alimentación:** El rubro más crítico bajo control.
* **Insumos Médicos:** Registro de vacunas, desparasitantes y consultas.
* **Mano de Obra:** Asignación de costos fijos y variables.
* **Cálculo Automático de Rentabilidad:** Conoce el margen neto por cada vaca.

### 🏥 Salud y Reproducción
* Control de ciclos de celo, inseminación y fechas probables de parto.
